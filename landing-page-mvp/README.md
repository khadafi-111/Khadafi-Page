# Personal Landing Page MVP

## Setup
1. Extract all files to your web server / folder
2. Replace `assets/images/profile.jpg` with your actual photo
3. Replace `assets/images/project1.jpg`, `project2.jpg`, `project3.jpg` with project screenshots
4. Edit `index.html` — search for "YourName" / "yourname@email.com" / placeholder links and fill in real info
5. Update social media links in Contact & Footer sections

## Folder Structure
```
landing-page-mvp/
├── index.html         ← Main page
├── css/style.css      ← All custom styles
├── js/script.js       ← Animations & interactions
├── assets/images/     ← Put your photos here
└── README.md
```
